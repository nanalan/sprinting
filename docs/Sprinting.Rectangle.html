new Rectangle([width][, height][, stroke][, fill][, strokeWidth])

Rectangles are Shapes that have a width and height.

Example 

// draws a 100x75 black-bordered rectangle in the World
let world = new World('#world')
world.add(new Rectangle).draw()

This method is called by the parent World's draw() method (although you can call it on it's own as shown in the example).
