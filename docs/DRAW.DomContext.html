DomContext

DomContext

new DomContext(world)

A DomContext is an inheritor of DrawingContext used for drawing with the DOM. It is automatically instanced when a new World is created with the usage World.USAGE_DOM. It has no new public attributes that it doesn't share with DrawingContext.
