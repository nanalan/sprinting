Sprinting

An amazing JavaScript library for drawing things with canvas. Bonus: use the DOM to debug and edit!
v0.0.1 super alpha

Installation

Include the script in your webpage:

<script src='//nanalan.github.io/sprinting/dist/sprinting.min.js'></script>
